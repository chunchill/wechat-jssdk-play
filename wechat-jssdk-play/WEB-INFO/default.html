活动介绍

首届DSM帝斯曼中国官方微信摄影大赛邀请所有员工和您的家属/朋友/客户/供应商及关注公司微信公众号的粉丝参加， 所有入围作品将在官方微信上展示/评选，大赛设立入围奖/人气奖/最佳主题奖/一二三等奖等奖项，以最终投票/评选结果为准。

投稿要求

  1. 照片以“缤纷科技，美好生活”为主题，相机/手机拍摄均可参赛，拍摄内容/时间/地点/黑白/彩色不限内容必须积极健康.
  2. 作品必须原创且未参加过其他任何比赛，作品如涉及肖像权/名誉权/著作权/版权/商标专利权等法律问题均由投稿人自行负责，主办方有权将参赛照片用于展览/宣传/交流/印刷等非商业使用。
  3. 照片一律以JPEG格式上传到官方微信（长边不超过1024像素），可适当后期处理但不能电脑合成，请保留照片原始尺寸以供进一步使用。

活动时间

  1. 2015年8月1日—2015年8月14日为活动推广期
  2. 2015年8月15日—2015年9月15日为作品上传/征集/展示期
  3. 2015年9月16日—2015年9月30日为入围作品初选期
  4. 2015年10月1日—2015年10月15日为入围作品投票/评选期
  5. 2015年10月20日公布获奖作品名单及奖品发放

投票及评选

  1. 每个ID每天只能投票一次
  2. 本次大赛将采取公众投票分数（占50%）和专家评审分数（占50%）综合得分评选
  3. 主办方保留对比赛活动的最终解释权
  4. 主办方：帝斯曼（中国）有限公司
